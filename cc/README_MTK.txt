CARA MENAMBAHKAN SOAL MTK SMP
=============================

File: insert_soal_mtk.php

Cara menggunakan:
1. Buka browser dan akses: http://localhost/cc/insert_soal_mtk.php
   atau
   http://your-domain/cc/insert_soal_mtk.php

2. Script akan otomatis memasukkan 50 soal MTK SMP ke database

3. Setelah selesai, akan muncul pesan sukses dengan jumlah soal yang berhasil dimasukkan

4. Soal sudah siap digunakan di halaman "Ikuti Lomba" â†’ MTK SMP

Topik Soal MTK SMP yang dibuat:
- Operasi bilangan bulat dan pecahan
- Aljabar dan persamaan linear
- Geometri (luas, keliling, volume)
- Statistika (rata-rata, median, modus)
- Perbandingan dan skala
- Sudut dan koordinat
- Fungsi dan grafik
- Faktorisasi dan persamaan kuadrat
- Peluang
- Barisan dan deret

Catatan:
- Pastikan koneksi database sudah benar di file insert_soal_mtk.php
- Jika ada soal duplikat, akan muncul error (bisa diabaikan)
- Setelah selesai, bisa hapus file insert_soal_mtk.php untuk keamanan
