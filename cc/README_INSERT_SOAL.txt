CARA MENAMBAHKAN SOAL IPA SMP DAN IPS SMP
==========================================

File: insert_soal.php

Cara menggunakan:
1. Buka browser dan akses: http://localhost/cc/insert_soal.php
   atau
   http://your-domain/cc/insert_soal.php

2. Script akan otomatis memasukkan:
   - 50 soal IPA SMP
   - 50 soal IPS SMP

3. Setelah selesai, akan muncul pesan sukses dengan jumlah soal yang berhasil dimasukkan.

4. Soal sudah siap digunakan di halaman "Ikuti Lomba"

Catatan:
- Pastikan koneksi database sudah benar di file insert_soal.php
- Jika ada soal duplikat, akan muncul error (bisa diabaikan)
- Setelah selesai, bisa hapus file insert_soal.php untuk keamanan
